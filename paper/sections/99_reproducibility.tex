This paper is reproducible from the repository with Python 3.12 and pinned dependencies:
\texttt{ortools==9.15.6755}, \texttt{PyQt6==6.10.2}, \texttt{python-docx==1.2.0}, and \texttt{pytest==9.0.2}.

\subsection*{Exact Experiment Pipeline}
All experiment batches and table files used in Section~7 are generated by:
\begin{verbatim}
PYTHON_BIN=.venv/bin/python ./scripts/run_experiment_batches.sh
\end{verbatim}

This command writes:
\begin{itemize}[leftmargin=*]
  \item baseline reliability/runtime runs: \texttt{paper/results\_baseline\_expanded.jsonl},
  \item local-search ablation runs: \texttt{paper/results\_ls\_expanded.jsonl},
  \item aggregated table rows: \texttt{paper/sections/07\_experiments\_table.tex},
  \item per-run rows: \texttt{paper/sections/07\_experiments\_runs.tex},
  \item local-search rows: \texttt{paper/sections/07\_experiments\_ls\_table.tex}.
\end{itemize}

\subsection*{Release Artifact Freeze}
For publication/release packaging, we freeze an artifact bundle with commit hash,
environment snapshot, checksums, and paper inputs:
\begin{verbatim}
PYTHON_BIN=.venv/bin/python ./scripts/freeze_release_artifacts.sh v0.2.0
\end{verbatim}

The output directory \texttt{release/v0.2.0/} contains:
\begin{itemize}[leftmargin=*]
  \item \texttt{environment.txt} (git SHA, Python version, package list, platform),
  \item \texttt{SHA256SUMS} (checksums of included files),
  \item copied paper/result inputs needed to rebuild tables.
\end{itemize}

\subsection*{Paper Build}
\begin{verbatim}
cd paper
latexmk -pdf -interaction=nonstopmode -halt-on-error main.tex
\end{verbatim}

CI validates both code and paper:
\begin{itemize}[leftmargin=*]
  \item Python checks in \texttt{.github/workflows/ci.yml},
  \item LaTeX+chktex checks in \texttt{.github/workflows/paper.yml}.
\end{itemize}
