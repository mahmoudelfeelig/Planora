To reproduce results:
\begin{itemize}[leftmargin=*]
  \item Record the Python version and package versions (OR-Tools, PyQt6, etc.).
  \item Record solver parameters (time limit, worker count, room mode, objective on/off, local-search iterations/time cap).
  \item Fix random seeds for instance generation.
  \item Save the generated instance and resulting schedule as a project snapshot (\texttt{.json} or \texttt{.pkl}).
\end{itemize}

For this paper, we recommend publishing:
\begin{itemize}[leftmargin=*]
  \item a table of all runs (seeds and parameters),
  \item the corresponding project snapshot files,
  \item the exact command lines used to run each experiment.
\end{itemize}
