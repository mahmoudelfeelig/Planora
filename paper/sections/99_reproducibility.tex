\subsection*{Exact Experiment Pipeline}
All experiment batches and table files used in Section~7 are generated by:
\path{./scripts/run_experiment_batches.sh}

This command writes:
\begin{itemize}[leftmargin=*]
  \item \path{paper/results_baseline_expanded.jsonl},
  \item \path{paper/results_ls_expanded.jsonl},
  \item \path{paper/sections/07_experiments_table.tex},
  \item \path{paper/sections/07_experiments_runs.tex},
  \item \path{paper/sections/07_experiments_ls_table.tex}.
\end{itemize}

The comparison diagrams in Section~7 use the same aggregated medians and counts
reported in these generated tables (no separate plotting data pipeline).

\subsection*{Release Artifact Freeze}
For publication/release packaging, we freeze an artifact bundle with commit hash,
environment snapshot, checksums, and paper inputs:
\path{./scripts/freeze_release_artifacts.sh v1.0}.

The output directory \path{release/v1.0/} contains:
\begin{itemize}[leftmargin=*]
  \item \path{environment.txt} (git SHA, Python version, package list, platform),
  \item \path{SHA256SUMS} (checksums of included files),
  \item copied paper/result inputs needed to rebuild tables.
\end{itemize}
