# Planora Brand Protection Checklist

1. Confirm the candidate mark is still available in target jurisdictions.
2. Search software, scheduling, and education product classes for conflicts.
3. Reserve matching domains and package names where possible.
4. Archive first public use evidence: screenshots, installer, release notes, exports.
5. Keep a controlled list of approved logos, colors, and product names.
6. Decide whether the mark is used as a product, platform, or service mark.
7. File the application only after the commercial name and scope are stable.
8. Use `TM` only if desired; do not use `R` unless registration is granted.
9. Track renewal deadlines and class coverage.
10. Maintain a simple infringement response workflow and evidence archive.
